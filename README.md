# End-to-End Insurance Risk Analytics & Predictive Modeling

## Project Overview
AlphaCare Insurance Solutions (ACIS) is aiming to optimize its marketing strategy and discover low-risk targets for premium reduction. This project involves analyzing historical insurance claim data to identifying low-risk segments and building predictive models.

## Structure
- `src/`: Source code for data processing and modeling.
- `notebooks/`: Jupyter notebooks for EDA and experimentation.
- `scripts/`: Executable scripts for pipelines.
- `tests/`: Unit tests.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Ensure the data file `MachineLearningRating_v3.txt` is in the project root (or update paths accordingly).
